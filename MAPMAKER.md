# Map Maker

How to make a Map.
Create a Text file with the suffix `.map`.
```
hello_world.map
```

What letters can I use?
| Letter | Description |
|--------|-------------|
|    #   | Walls       |
|    .   | Goals       |
|    $   | Box         |
|    @   | Player      |


## Rules
1. In one map can only be one Player.
2. For each Goal there need to be a Box
3. The map should not let the player leave the map.
